# Интерактивный Задачник с AI-Помощником

## Описание проекта
Создан интерактивный задачник, который позволяет:

- Выбирать задания из учебника.
- Вводить ответы на задания.
- Проверять правильность ответа.
- Получать правильный ответ в случае затруднения.

Особенности:

- Возможность просмотра решения **без необходимости заглядывать в конец книги**.
- Автоматическая обработка учебников: вырезка задач и ответов, сортировка по папкам.
- Легко масштабируемая система, подходящая для учеников и учителей.

---

## Используемые технологии

### Для обработки данных и изображений
- `fitz` – работа с PDF
- `pytesseract` – OCR для распознавания текста
- `cv2` (OpenCV) – обработка изображений
- `numpy` – работа с массивами
- `os`, `re` – файловая и текстовая обработка

### Архитектура приложения
- **C#** – основной каркас приложения
- **Nemotron‑3‑Nano‑30B‑A3B (NVIDIA)** – LLM с режимом рассуждения
- AI-помощник для выдачи теории и подсказок
- Возможность развёртывания под сервер

---

## Преимущества
- Эффективная помощь ученикам без спойлеров.
- Автоматизированная обработка учебников снижает ручную работу.
- Масштабируемость и удобство интеграции в образовательные процессы.
- Подходит для развертки как на локальном сервере, так и в виде приложения.
